# Taskly - MERN Task Manager

A modern Task Management application built with the MERN Stack.

## Features

- User Authentication (JWT)
- Register & Login
- Protected Routes
- Create Tasks
- Edit Tasks
- Delete Tasks
- Mark Complete/Pending
- Search Tasks
- Responsive Design
- Modern Glassmorphism UI
- Toast Notifications

## Tech Stack

Frontend
- React
- Tailwind CSS
- Axios
- React Router
- Framer Motion
- React Hot Toast

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

## Installation

Frontend

npm install
npm run dev

Backend

npm install
npm run server
